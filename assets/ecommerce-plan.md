# Ecommerce SEO Plan

## Priority Pages

- main categories
- subcategories
- top product pages
- brand pages
- shipping and returns
- buying guides

## Main Priorities

1. top categories;
2. best-selling products;
3. brand and collection pages;
4. buying guides.

## Main Schema

- `Product`
- `Offer`
- `AggregateRating`
- `BreadcrumbList`
- `ProductGroup` for variants

## Main Risks

- faceted duplicate pages;
- thin product pages;
- copied manufacturer text.

## Main Metrics

- organic revenue;
- category growth;
- product page growth;
- search result click-through rate.
