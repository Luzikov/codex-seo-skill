# SaaS SEO Plan

## Priority Pages

- homepage
- product
- features
- integrations
- pricing
- case studies
- competitor comparisons
- docs

## Main Priorities

1. feature and use-case pages;
2. integration pages;
3. comparison pages;
4. proof pages: case studies and testimonials;
5. helpful articles around real buyer questions.

## Main Schema

- `Organization`
- `WebSite`
- `SoftwareApplication` or `WebApplication`
- `Person`

## Main Metrics

- visits to pricing;
- demo or trial signups;
- growth of integration pages;
- growth of comparison pages.
