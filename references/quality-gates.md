# Quality Gates

## Minimum Content Benchmarks

| Page Type | Min Words | Unique Content |
|---|---:|---:|
| Homepage | 500 | 100% |
| Service or Feature | 800 | 100% |
| Main Location | 600 | 60% or more |
| Secondary Location | 500 | 40% or more |
| Blog Post | 1500 | 100% |
| Product Page | 400 | 80% or more |
| Category Page | 400 | 100% |
| About Page | 400 | 100% |
| Landing Page | 600 | 100% |
| FAQ Page | 800 | 100% |

## Title Tag

- usually best at 30-60 characters;
- put the main topic early;
- keep each title unique;
- avoid stuffing keywords.

## Meta Description

- usually best at 120-160 characters;
- explain the benefit clearly;
- avoid duplicate descriptions.

## Image Alt Text

- required for non-decorative images;
- usually 10-125 characters;
- describe the image, do not list keywords.

## Internal Links

| Page Type | Normal Target |
|---|---|
| Blog post | 5-10 internal links |
| Service page | 3-5 internal links |
| Category page | links to child pages |
| Product page | 2-4 internal links |

## Location Page Limits

Use these rules to avoid doorway-page risk:

- warn at `30+` location pages;
- hard-stop at `50+` location pages without a strong business reason;
- each page must contain real local details, not just a swapped city name.

Risk signs:

- only city names change;
- no local reviews or proof;
- no real business presence in that place;
- keyword-heavy URLs and headings.

Safer scaled pages:

- integration pages;
- glossary pages;
- tool pages;
- product pages with real differences.
