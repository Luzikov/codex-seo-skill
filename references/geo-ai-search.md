# AI Search Readiness

Use this file for ChatGPT, Perplexity, AI Overviews, and related visibility reviews.

## Bots To Check

| Bot | Typical Use |
|---|---|
| GPTBot | OpenAI training crawler |
| OAI-SearchBot | OpenAI search crawler |
| ChatGPT-User | page retrieval for user browsing |
| ClaudeBot | Anthropic crawler |
| PerplexityBot | Perplexity crawler |
| CCBot | Common Crawl |

If important bots are blocked, say which ones are blocked and where.

## Signals That Help AI Citation

- short self-contained answer blocks;
- clear definitions near the top of a section;
- tables, lists, and comparisons;
- facts, numbers, and sources;
- clear authorship;
- visible content without heavy client-side JavaScript.

## llms.txt

`/llms.txt` is optional, but useful as a simple guidance file for AI systems.

If missing, you may suggest a minimal template:

```text
# Site Name
> Short site description

## Important pages
- https://site.com/
- https://site.com/pricing
- https://site.com/blog
```

## Explain It Simply

- Traditional SEO helps a page get found.
- AI search readiness helps a page get quoted in an answer.
- A page can be indexed and still be poor at AI citation.

## Safety

- do not promise one change will unlock AI traffic;
- do not promise exact growth without live data;
- do not advise blocking all AI bots if the user's goal is AI visibility.
